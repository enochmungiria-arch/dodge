## variables

*This is where we allocate a particular value to something*

### For example:
x=35
y=12
z=x+y
In Javascript we declare variables by using:
 - let 
 - const
 - 
 We should avoid using var to declare variables
 

### The rules to consider when using variables are:

 - Variables are very case sensitive 
 for example: let_me _pay and Let_me_pay are different variables
 
 - Spacing should not be used.
 - Reserved words should not be used in naming in Javascript.
 - Only letters, digits, dollar signs, and underscores are permitted in Javascript.
 - Alphabets, underscores(_), dollar signs($) are permitted to be used as the first character in a variable. No other special characters can be used.


## Primitive variables
the primitive variables are :

 - Numbers
 - String
 - Boolean
 - Null
 - Undefined

Numbers
Represents integers and floating types of data

 1. let myFiles=2 //integer value
 2. const myBaalance=32.55 //decimal (floating)
 
 String
 It is usually a sequence of characters
 The string value is usually surrounded by single quotes, double quotes or 
 
 1. let myCar= "Brabus"
 2. let myPet= 'Mick'
 3. let theToy=`Polarbear`(``)
 
 Boolean
 The values are either true or false
 let doYouHaveMoney=true
 let wereYouPaid=false

Undefined
Never set the variable to undefined. Javascript adds the variable when left undefined

Null
Null means the absence of a variable. You can explicitly set the variable
let grade=null //input grades
